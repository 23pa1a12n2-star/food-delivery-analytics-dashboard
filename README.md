# food-delivery-analytics-dashboard

## 📊 Project Overview
This project provides an end-to-end data analysis of food delivery operations. The goal of this dashboard is to derive actionable insights regarding market performance, revenue distribution, operational efficiency, and customer preferences.

## 🚀 Key Visualizations
This dashboard provides a multi-dimensional view of the business:

*   **Market Presence:** Analyzes order volume by city to identify key growth areas.
*   **Financial Performance:** Breaks down revenue contribution by meal type (Breakfast, Lunch, Dinner, Evening Snacks).
*   **Operational Efficiency:** Visualizes delivery time distribution to identify bottlenecks in the supply chain.
*   **Cuisine Popularity:** A detailed grid analysis showing cuisine preferences across different geographical locations.

## 🛠 Tech Stack
*   **Data Visualization:** Tableau Public
*   **Data Processing:** Data Cleaning & Exploratory Data Analysis (EDA)

## 📈 Insights
*   Identified peak delivery windows to optimize logistics.
*   Mapped high-demand cuisines to specific cities, aiding in localized marketing strategies.
*   Evaluated meal-type profitability to support inventory/menu planning.
